# Go Learning

## Module 1: Go Foundations (Week 1)

**Goal:** Get Go installed, understand syntax, variables, and control flow.

### Core Concepts

- Installing Go & “Hello World”
- Variables, Constants, and Data Types
- Operators & Expressions
- Conditional Statements (if, switch)
- Loops (for)

### Bite-Sized Projects

- Project 1: Hello Go!

  Print “Hello, Go World!” and system information using runtime package.

- Project 2: Temperature Converter

  Convert Celsius to Fahrenheit using variables and user input.

- Project 3: Even/Odd Classifier

  Take user input and classify numbers as even/odd using conditionals.

- Project 4: FizzBuzz Go Style

  Implement FizzBuzz using for loops and conditionals.

### Mid-Sized Project

- CLI Calculator

  Build a simple command-line calculator that performs +, -, ×, ÷ using user input, variables, and control flow.
